# Template: Daily Agent Brief

Use this when asking an AI assistant to make steady progress on a public or synthetic project.

## Goal

Make 1-3 small improvements today.

## Allowed Work

- public documentation
- synthetic examples
- templates
- checklists
- eval scenarios
- tests and scans
- verification receipts

## Hard Stops

Stop before:

- submitting external forms
- changing account settings
- touching billing
- creating or exposing API keys
- enabling data sharing
- using private repository content
- using raw transcripts
- using production logs or configs
- using wallet, session, auth, or billing data
- using user or customer data

## Daily Steps

1. Pick 1-3 tasks.
2. Say what will be changed.
3. Make the smallest useful edit.
4. Run relevant checks.
5. Write a short receipt.
6. Stop and report hard stops or questions.

## Receipt

- tasks completed:
- files changed:
- checks run:
- public repo update:
- hard stops:
- next tasks:
