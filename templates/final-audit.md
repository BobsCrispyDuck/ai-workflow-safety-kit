# Template: Final Audit

Use before saying a larger goal is complete.

## Audit Questions

1. What was the original objective?
2. What concrete requirements came from it?
3. What evidence proves each requirement?
4. What checks were run?
5. What remains unverified?
6. What still needs approval?
7. Did any action cross an external or irreversible boundary?

## Completion Rule

Do not mark complete if:

- proof is missing
- only part of the objective is done
- a required approval is missing
- the result has not been checked against current files or runtime state
- completion depends on an assumption

## Receipt Snippet

```text
Final audit:
- objective:
- requirements checked:
- evidence:
- complete: yes/no
- unresolved items:
- approval still needed:
```
