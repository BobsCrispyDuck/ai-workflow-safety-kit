# Checklist: Before Editing

- Confirm the project root.
- Confirm the file belongs to the task.
- Confirm the path is not production.
- Confirm the path is not a synced/archive copy.
- Check whether the work needs private data.
- Check whether the change affects users, billing, auth, wallet/session state, account settings, or deployment.
- Identify the verification command or evidence before editing.
- Stop if approval is required.
