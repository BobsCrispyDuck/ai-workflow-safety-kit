# Checklist: Before Submitting

- Identify the destination.
- List every field that will be submitted.
- Remove private data.
- Remove secrets, keys, logs, transcripts, account IDs, and user data.
- Confirm examples are public or synthetic.
- Confirm the source program is still active.
- Confirm the requested credit type.
- Run a private-data scan.
- Prepare an approval packet.
- Get explicit approval.
