# Starter Pack

If you want to borrow this kit, do not copy the whole repo first.

Start with the smallest pack that matches the work.

## Minimum Pack

Use this before a normal AI-assisted coding session.

Copy:

- `templates/work-session-check.md`
- `templates/project-root-check.md`
- `templates/private-public-data-split.md`

Use when:

```text
I want an assistant to work in this repo, but I need it to stay inside the right folder and avoid private material.
```

## Public Release Pack

Use this before anything leaves your machine.

Copy:

- `checklists/before-publishing.md`
- `templates/public-release-check.md`
- `templates/verification-receipt.md`
- `docs/false-positives-and-misses.md`

Use when:

```text
I want to publish a doc, example, repo update, screenshot, release note, issue, or post.
```

## Submission Pack

Use this before forms, applications, support notes, community posts, or anything else that gets sent to a third party.

Copy:

- `checklists/before-submitting.md`
- `templates/approval-gate.md`
- `templates/verification-receipt.md`

Use when:

```text
I want to draft locally, list exactly what would be sent, and stop before submitting.
```

## Review Pack

Use this if you want someone else to test the kit quickly.

Copy:

- `docs/reviewer-prompts.md`
- `docs/reviewer-scorecard.md`
- `docs/review-feedback-examples.md`

Use when:

```text
I want one useful public-safe note, not a big report.
```

## Shared-Token Or Public/Synthetic Pack

Use this only when prompts and outputs may be visible outside your private workspace.

Copy:

- `checklists/before-using-shared-tokens.md`
- `templates/private-public-data-split.md`
- `docs/model-use-plan.md`

Use when:

```text
I want to keep cheap or shared lanes limited to fake examples and public docs.
```

## After The Work

Use these before calling a bigger task done:

- `templates/verification-receipt.md`
- `templates/final-audit.md`
- `docs/example-result-receipts.md`

Done should mean:

- files changed are named
- checks are named
- skipped checks are named
- remaining approval is named
- the current state was actually verified
