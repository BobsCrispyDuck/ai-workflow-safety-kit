# Why This Exists

AI coding agents make it easier to build, but they also make it easier to move too fast.

Many mistakes are not complicated engineering failures. They are workflow failures:

- editing the wrong folder
- mixing private and public context
- publishing before review
- changing production without approval
- accepting "done" without evidence

This kit gives builders small, reusable guardrails for those moments.

It is intentionally simple. The point is not to create process theater. The point is to make the next safe action obvious.
